# Food Delivery Web Application

## Overview
This project is a frontend food delivery web application that allows users to browse food items, view menus, and explore a responsive restaurant-style interface.

## Features
- Responsive food menu layout
- Interactive UI components
- Navigation between sections
- Modern food ordering interface

## Technologies Used
HTML5  
CSS3  
JavaScript

## Learning Outcomes
- Responsive web design
- Layout structuring using CSS
- User interface development
- JavaScript interaction handling
